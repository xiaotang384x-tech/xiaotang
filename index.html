XBotGO APP
视觉与交互体验升级

UI/UX Design App Redesign 2023

项目图片展示

01 项目背景

XBotGO 是一款专注于智能硬件控制与数据监控的移动端应用。随着硬件产品的迭代，旧版 APP 在视觉语言、操作逻辑和信息层级上逐渐显露出不足，无法满足核心用户对于"高效、专业、科技感"的期待。

核心痛点：设备连接状态不直观、控制面板信息过载、整体视觉风格陈旧缺乏品牌识别度。

设计目标：重构核心操作链路，建立清晰的信息层级，打造现代科技感的全新视觉规范。

架构分析：采用模块化分层架构，将设备控制、数据展示、用户设置三大核心模块解耦。通过状态管理中台实现跨模块数据同步，确保设备联动响应实时性。

02 解决方案

1. 视觉语言重塑

提取品牌基因中的科技蓝作为主色调，结合深色模式打造沉浸式控制体验。采用大圆角卡片与毛玻璃效果，提升界面的现代感与呼吸感。

2. 信息架构优化

将原本散落的控制项进行场景化聚合，通过底部抽屉和手势滑动，让单手操作更加流畅。核心功能触达路径从平均 3 步缩短至 1.5 步。

3. 交互体验升级

引入实时状态反馈机制，设备连接、指令下发、结果确认形成完整闭环。微动效设计强化操作反馈感，降低用户焦虑。

03 我的任务

工作范围

  • 参与前期用户访谈与需求分析
  • 重构设备控制核心页面的信息架构
  • 制定全新的 UI 视觉规范体系
  • 输出高保真设计图及交互动效 demo

核心角色

  • 主导视觉升级的主干设计师
  • 跨部门沟通（产品/研发）的桥梁
  • 设计规范的建立者与维护者

04 需求挖掘

通过对 50+ 核心用户的深度访谈以及针对竞品（如 DJI, GoPro 等）的横向分析，我们发现用户在实际操作中，最关注的是"当前状态感知"与"核心功能触达效率"。

[ 用户旅程图 / 痛点分析图表 ]

05 设计执行

视觉规范：提取品牌基因中的科技蓝作为主色调，结合深色模式打造沉浸式控制体验。采用大圆角卡片与毛玻璃效果，提升界面的现代感与呼吸感。

交互优化：将原本散落的控制项进行场景化聚合，通过底部抽屉和手势滑动，让单手操作更加流畅。

[ 高保真界面展示 / 视觉规范画板 ]

06 结果验证

新版本上线后，核心功能的触达路径从平均 3 步缩短至 1.5 步。通过可用性测试，用户任务完成率提升了 35%。

斗米 APP
核心链路重构

Product Design UX Research 2024
[ 斗米项目主视觉图 ]

01 项目背景

斗米是国内领先的灵活用工服务平台。随着业务的扩张，原有的找兼职、报名、面试等核心链路变得臃肿，用户流失率增加。

设计目标：简化求职者从"发现岗位"到"确认上岗"的全链路体验，降低认知门槛，提升转化率。

02 我的任务

工作范围

  • 主导核心求职链路的交互梳理
  • 设计品牌吉祥物的全新视觉
  • 更新并推行全端设计规范

核心角色

  • 体验设计负责人
  • 用户研究调研员

03 需求挖掘

通过数据埋点分析发现，用户在"填写简历"和"职位筛选"环节流失最严重。我们将其转化为具体的体验优化指标：减少必填项、增加智能推荐。

[ 数据漏斗图 / 调研报告 ]

04 设计执行

体验重塑：引入分步式简历填写，利用卡片式设计隔离信息噪音。同时，将品牌吉祥物巧妙融入加载、空状态等环节，提升情感化体验。

[ 核心页面对比图 / 吉祥物应用 ]

05 结果验证

改版后，简历完整度提升显著，核心岗位的报名转化率实现了大幅度增长。

《斗米大富翁》
活动游戏化设计

Game UI Event Design Illustration
[ 大富翁活动主视觉图 ]

01 项目背景

为提升平台活跃度与用户粘性，结合节假日节点推出《斗米大富翁》线上互动游戏。希望通过游戏化手段（Gamification）促进用户每日签到与分享。

02 我的任务

工作范围

  • 游戏地图与建筑元素的插画绘制
  • 游戏核心界面的 UI 设计
  • 交互动效与反馈机制设计

核心角色

  • 主视觉插画师
  • 游戏化 UI 设计师

03 需求挖掘

分析了市面上主流的轻量级 H5 游戏，提炼出"即时反馈"、"阶段性奖励"和"社交炫耀"三个核心驱动力，作为设计的基调。

[ 竞品分析 / 玩法机制图 ]

04 设计执行

视觉呈现：采用 2.5D 轴测图视角绘制地图，色彩明快活泼。设计了丰富的微动效（如掷骰子、金币掉落），强化游戏的沉浸感。

[ 地图元素展示 / 动效演示 ]

05 结果验证

活动上线期间，DAU 显著提升，用户在端内的平均停留时长翻倍，社交分享率超出预期。

02 我的任务

工作范围
